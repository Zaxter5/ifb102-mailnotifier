TO RUN THIS PROGRAM DOWNLOAD main_final.py AND STORE ON YOUR RASPBERRY PI. 
IN THE SAME FOLDER YOU MUST DOWNLOAD YOUR OWN client_secret.json BY FOLLOWING THESE STEPS https://developers.google.com/gmail/api/quickstart/python AND PLACING IT IN THE SAME FOLDER AS main_final.py. 
IN THE EDIT MENU, ADD YOUR CERTIFIED EMAIL. 
ATTACH AN LED WITH 330OHM RESISTOR TO GPIO PIN 17 FOR LED TO WORK.
HOOK UP SPEAKER TO 3.5MM JACK ON RPI. 
ENJOY!
